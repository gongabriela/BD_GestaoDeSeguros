Dicionário de Dados - Gestão de Seguros

Este documento detalha a estrutura do banco de dados GestaoDeSeguros, incluindo tabelas, colunas, tipos de dados e relacionamentos.

Tipos de Dados Personalizados

Este banco de dados utiliza tipos de dados personalizados para garantir a consistência e a integridade dos dados.

Dom_Moeda

Descrição: Tipo de dado para representar valores monetários.
Tipo Base: DECIMAL(10, 2) NOT NULL.
Regra Associada: RL_ValoresMonetarios, que garante que o valor seja sempre maior ou igual a zero (>= 0).
Uso: Utilizado em todas as colunas que armazenam valores financeiros, como prémios, franquias e indemnizações, para assegurar que não sejam negativos e que tenham sempre duas casas decimais.

EntidadePessoa

Descrição: Armazena informações sobre as entidades (pessoas físicas ou jurídicas) que são clientes ou potenciais clientes.

Coluna Tipo de Dados Restrições Descrição
EntidadePessoaID SMALLINT PK, IDENTITY(1,1) Identificador único da entidade.
Nome VARCHAR(100) NOT NULL Nome completo ou razão social da entidade.
NIF CHAR(9) NOT NULL, UNIQUE Número de Identificação Fiscal da entidade.
TipoEntidade VARCHAR(20) NOT NULL, CHECK Tipo de entidade ('Pessoa Física' ou 'Pessoa Jurídica').
DataNascimento DATE NOT NULL Data de nascimento ou de constituição da entidade.
Morada VARCHAR(200) NOT NULL Endereço da entidade.
Localidade VARCHAR(100) NOT NULL Localidade da morada.
Telefone CHAR(15) NOT NULL Número de telefone para contato.
Email VARCHAR(100) NOT NULL Endereço de e-mail para contato.
Status VARCHAR(10) NOT NULL, CHECK Status da entidade ('Ativo' ou 'Inativo').
Observacoes VARCHAR(200) NULL Observações adicionais sobre a entidade.

EntidadeBem

Descrição: Armazena informações sobre os bens segurados (e.g., imóveis, veículos) pertencentes a uma entidade.

Coluna Tipo de Dados Restrições Descrição
EntidadeBemID INT PK, IDENTITY(1,1) Identificador único do bem.
ProprietárioID SMALLINT NOT NULL, FK to EntidadePessoa Identificador do proprietário do bem.
DescricaoBem VARCHAR(200) NOT NULL Descrição detalhada do bem.
IdentificacaoBem VARCHAR(100) NOT NULL, UNIQUE Identificação única do bem (e.g., matrícula, chassi).
ValorEstimado Dom_Moeda NOT NULL Valor estimado do bem.
DataAquisicao DATE NOT NULL Data de aquisição do bem.
Observacoes VARCHAR(200) NULL Observações adicionais sobre o bem.

Localidade VARCHAR(100) NOT NULL Localidade da morada.

MediadorSeguradora

Produto

Descrição: Armazena os produtos de seguro oferecidos pelas seguradoras.

Coluna Tipo de Dados Restrições Descrição
ProdutoID SMALLINT PK, IDENTITY(1,1) Identificador único do produto.

ProdutoID SMALLINT NOT NULL, FK to Produto Identificador do produto ao qual o plano pertence.

DataFimVigencia DATE NULL Data de fim de vigência do plano.
Observacoes VARCHAR(200) NULL Observações adicionais.

Apolice

Descrição: Armazena as apólices de seguro contratadas.

Coluna Tipo de Dados Restrições Descrição
ApoliceID INT PK, IDENTITY(1,1) Identificador único da apólice.
ProdutoPlanoID INT NOT NULL, FK to ProdutoPlano Identificador do plano de seguro contratado.
MediadorID SMALLINT NOT NULL, FK to Mediador Identificador do mediador envolvido.
DataEmissao DATE NOT NULL, DEFAULT GETDATE() Data de emissão da apólice.
DataInicioVigencia DATE NOT NULL Data de início de vigência da apólice.
DataFimVigencia DATE NOT NULL Data de fim de vigência da apólice.

ApoliceEntidadePessoa

Descrição: Associa as entidades (pessoas) a uma apólice, definindo o seu papel.

Coluna Tipo de Dados Restrições Descrição
ApoliceEntidadeID INT PK, IDENTITY(1,1) Identificador único da associação.
ApoliceID INT NOT NULL, FK to Apolice Identificador da apólice.
EntidadePessoaID SMALLINT NOT NULL, FK to EntidadePessoa Identificador da entidade.
Papel VARCHAR(15) NOT NULL, CHECK Papel da entidade na apólice ('Tomador', 'Segurado', 'Beneficiário').

ApoliceEntidadeBem

Descrição: Associa um bem segurado a uma apólice.

Coluna Tipo de Dados Restrições Descrição
ApoliceEntidadeBemID INT PK, IDENTITY(1,1) Identificador único da associação.
ApoliceID INT NOT NULL, FK to Apolice Identificador da apólice.
EntidadeBemID INT NOT NULL, FK to EntidadeBem Identificador do bem.

HistoricoApolice

Descrição: Regista o histórico de alterações de estado de uma apólice.

Coluna Tipo de Dados Restrições Descrição
HistoricoApoliceID INT PK, IDENTITY(1,1) Identificador único do registo de histórico.
ApoliceID INT NOT NULL, FK to Apolice Identificador da apólice.
DataAlteracao DATETIME NOT NULL, DEFAULT GETDATE() Data e hora da alteração.
Estado VARCHAR(15) NOT NULL, CHECK Novo estado da apólice ('Ativa', 'Suspensa', 'Cancelada', 'Expirada').

Coluna Tipo de Dados Restrições Descrição

Coluna Tipo de Dados Restrições Descrição
